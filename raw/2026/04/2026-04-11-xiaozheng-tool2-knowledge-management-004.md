## Raw Content

**【工具二】：知识管理**

本着**『什么能帮你赚到钱，我们就做什么』的原则，**在社群初创之时，我收到了大量的反馈说有学习有干货社群氛围好，但就是**感觉很厉害，实际自己没有成长。**

**做个简单的调研**

1、喜欢看书吸收知识的

2、喜欢看视频/短视频吸收知识的

3、喜欢通过实践收获知识的

4、对知识不太感冒的

  ----------------------------------------------------------- ------------------------------------------------------------
     
  ----------------------------------------------------------- ------------------------------------------------------------

经过和部分伙伴沟通和调研，我们发现大部分人，**没有构建过属于自己的知识体系**，一直在短视频和各种公众号文章里徘徊，今天转发一个感觉不错的，明天点赞一个感觉有用的；**甚至常见于经常喜欢收藏一些视频，但是从来就再也不看。**

所以，正式推出**《知识管理》工具课程1.0**，希望能够帮助到渴望真正成长和进步的你。

+-----------------------------------------------------------------------------+
| **个人知识的管理，对于这四类情况非常关键：**                                |
|                                                                             |
| **1、希望成为专家：**在某一些领域/话题上不断精进，成为专家。                |
|                                                                             |
| **2、希望超强IPO：**构建自己的IPO知识模型，支撑自己快速阅读和消化内容。     |
|                                                                             |
| **3、希望持续输出：**拥有持续输出能力，比如持续写作、写脚本、做课、做创意。 |
|                                                                             |
| **4、希望长期变强：**以年为单位，打造一个某些能力超强的自己。               |
|                                                                             |
| 只要你满足1-2条，那知识管理就是你绕不过去的必经之路。                       |
+-----------------------------------------------------------------------------+

**在社群发展的过程当中，我们发现大家通常会发现有以下三个困扰**

+--------------------------------------------------------------------------------------------------------------------------------------------------------+
| **困扰1：缺少知识方面的"审美"，不知道什么才是好的知识管理方法。**                                                                                      |
|                                                                                                                                                        |
| 特别渴望去做知识管理，但是没有见过谁有真正好的案例，对有能力做知识管理的人来说，自己的知识库是一件很私密的事情。所以市场上内容稀缺，只能自己凭感觉做。 |
|                                                                                                                                                        |
| **困扰2：信息量过大过密，大量丢失。**                                                                                                                  |
|                                                                                                                                                        |
| 平时遇到的信息量太大了，大量的文章、开会、课程、朋友圈金句、社群聊天等涌过来，感觉有很多精华，但是没有一个很体系的方法或者工具记下来，长期去复用。     |
|                                                                                                                                                        |
| **困扰3：收藏很多，用得太少。**                                                                                                                        |
|                                                                                                                                                        |
| 平时收藏了大量的文章，写了大量碎片笔记，但是一直没有什么用，积累的越多，内容就越乱，压力越大，反而就不想要了，**收藏=学了=废了。**                     |
+--------------------------------------------------------------------------------------------------------------------------------------------------------+

这些困扰连在一起，会导致很多人可能**做了10分的事情**，只**沉淀下来3分**，最终能够**调用的变成1分**。

你们见到的大部分在社群一发言就让你们醍醐灌顶的人，或者你觉得他们某一个能力特别强的人，都在做自己的知识管理，并且以知识管理为起点，构建了自己现在的能力壁垒。

**一、预热思考：**

大家平时可能会开很多的会、学很多的课、看很多的书或者文章、有很多有启发的案例或者思考，你会把它们全部收集在一起吗？**会的扣1，不会的扣2**

**能留存1年吗？还是3年、10年？需要用的时候，你能够快速调用吗？**

你平均每一年，会产生多少篇新的笔记或者知识内容？

什么是好的知识管理，**你觉得有哪些要素？怎么样算合格？怎么样算优秀？**你自己有追求一种丝滑的、高效的、长期的知识管理系统吗？

**这两个问题，都不简单；**跟着小林的节奏，希望这堂课程结束，你在审美、知识管理原则和技巧的方面，能有一个全面的提升。

**二、课程目录：**

**第一个内容：建立你自己的知识审美**

我会给大家讲讲我自己真实的案例，**看一下真实的知识管理系统的形成、思考及过程**，大家可以感受一下，背后的审美和思考原则。

**第二个话题：马上成长**

**小挣青年承诺：不学没用的，不浪费时间，**学完马上落地实践，看看【合格】→【进阶】→【高阶】分别是什么水平，以及如何掌握背后的方法和执行技巧。

**最后一部分：为什么知识管理是加速财富收集的法宝？**

知识管理和财富之间的存在的必然联系，如果你看不到，你就无法科学的去抓住这笔财富，只能靠自己的直觉做判断。

**三、知识管理的各个阶段**

**第一阶段：靠脑子记**

很多人在读书的时候就知道，记笔记是一个非常好的习惯，然后因为这个习惯让自己在学习成绩上取得了正反馈，所以在进入职场和社会之后，依然会保留这样的习惯，开会一定要带个纸笔，还有笔记本电脑。实在很紧急没带东西最起码也会用手机来记。

当时记录的时候，感觉很爽，然后经过一段时间的记录之后，应该会出现这样的问题：

**第一个问题：我发现真正需要用的时候，这些东西，我找不到了。**

会上跟领导说对对对，会后跟领导说我不会

**第二个问题：当我换工作之后，这些东西，也消失了，我不知道自己积累了什么。**

工作哪怕在努力，取得了不少成绩，也积累了几个笔记本，但是就是感觉自己没有获得**很明显的进步感和成长感。**

大家有这个感受的，**扣个1；**很多人都是这个阶段开始的，如果说你从来没有做笔记的习惯，你从来都是靠直觉，你依赖你对社会的体感和直觉，你是非常容易出现问题的，而且容易陷入一种偏执的状态，太过于绝对了。

每个真正的数字游民和自由职业者，只要是追求长期发展的，阶段都是从这个时候开始的；但是有些人很快就能过渡到下一个阶段，有些人则会在这个阶段停留很久。

**第二阶段：记下来备忘**

从第一个阶段到第二个阶段，就好像原始人用石头当武器，变成了制造刀剑开始开辟道路。

在这个阶段最重要的标志，就是**开始【借助工具】。**

比如印象笔记，飞书，语雀，腾讯文档、**微信文件传输助手**乃至手机备忘录或者纸和笔等等工具。

很多方案，很多思考，很多自己一眨眼看到的金句，一篇一篇往里面丢。

  --------------------------------------------------------------------- --------------------------------------------------------------------- ----------------------------------------------------------- ------------------------------------------------------------------------------------
           
  --------------------------------------------------------------------- --------------------------------------------------------------------- ----------------------------------------------------------- ------------------------------------------------------------------------------------

好处显而易见，**我不太会错过身边优质或者重要的信息内容了。**

我从很早之前就很喜欢看知乎和公众号长文，经常会看到很多优秀的文案和让人灵感迸发的好文，但是那个时候看完就诶嘿一下，也不记录；一部分原因是没有这个意识，另外一个原因是没有很趁手的工具，实在觉得好的不得了我就转发丢到文件传输助手或者收藏，然后也没有然后了。

**后来，我用腾讯文档做了一个属于自己的【想到什么说什么记录表格】**

  ------------------------------------------------------------------------------------ ------------------------------------------------------------------------------------
     
  ------------------------------------------------------------------------------------ ------------------------------------------------------------------------------------

这个文档，承载了我当时很多的思考和记录，以至于当我真正开始做自媒体的时候发现，**很多人在强调的选题库，不就是我这个玩意嘛？**

所以你们B站上面看到的很多内容，其实都**来自于我这篇【想到什么记什么】**

在这个阶段，我不太容易丢重要的记录，而且我用的腾讯文档，只要记得关键词就可以直接搜索。

  ------------------------------------------------------------------------------------
  **直到最近一两年，我开始真正想要制作一套自己的课程内容时，我发现了一个新的问题。**
  ------------------------------------------------------------------------------------

随着我的内容越来越多，所有的笔记都是直接这样排序的，有骂人的，有选题的，有思考的，有创业的，有业务的。东一块，西一块。

**当我的东西越来越多的时候，有些东西，又开始找不到了，或者说，不知道放哪里合适了。**

**比如一篇营销相关的感悟，我是放在【感悟】还是放在【营销】里？**

**比如一篇读书相关的思考，我是放在【读书】还是放在【感悟】里？**

这个时候，我就开始思考，**如何让自己的知识开始变得【体系化】**，这个意识就好像种了一棵树，然后每一个知识都是一个分枝。

于是我就开始了新的应用方法。

**第三阶段：借助工具完成【知识IPO】初建**

有时候可能不是一篇文章或者一本书所有的内容都会让我们有所感悟，更多是某一句话或者某一个案例打动了我们，我在23年下半年预估做了300多篇自己喜欢的内容整理，300多篇也就意味着我至少看过上千篇其他内容，沉淀下来的仅有这300多篇。

  ------------------------------------------------------------------------------------ -------------------------------------------------------------------------
     
  ------------------------------------------------------------------------------------ -------------------------------------------------------------------------

也是在那个时候，我完成了我自己的**【首次知识升级和积累：衣柜的第一次整理】**没有这次的积累，我做不到后面的很多输出，比如小红书的起号，比如B站的起号，比如社群的运营等等；是这些知识推动着我往前走，而且很自然的推动。

这个阶段对我来说，帮助非常大，最显著的成果就是我靠着这种思考模板，**在自媒体做内容方面不会特别困难，非常丝滑。**

|                                                                                                                  |
| 收藏的内容开始超过了我的吸收速度，因为只要我觉得有意义，我就收藏了，收藏太多导致我没有办法快速去吸收这些内容了。 |

**我就开始担心，这个操作是不是我真的解决了自己的问题？我是不是真的一直在成长？**

这个阶段就好像你辛苦的造了一辆车，然后因为功能太多了变成了飞机或者火箭，你已经完全驾驭不了了。

很多人因为短视频时代，养成了吸收短平快知识的特质，但一直忽略了**知识本质上是需要沉淀**的，也就导致了一个后果：**看的东西越来越多，但做事能力几乎没有增长。**

**俗称：高认知穷人。**

同时，因为我们处于这种状态之下时注意力被无限分散，这个时候一旦察觉自己没有增长就会陷入焦虑的情绪。

就像纸上谈兵的理论一样，你遇到具体的问题的时候，你发现你学的知识毫无用处，能够调用的异常有限；更不要提用知识帮你赚钱了。压根不可能。

所以，我开始继续去思考：**我的衣柜里，哪些是真正有效的东西？**

**第四个阶段：知识树的创立（消化）**

到了这个阶段，我开始反问自己：**我想要什么，不想要什么？**

**怎么去判断，我想要什么？或者说，我怎么确定，这个我真的需要？**

你有没有遇到过这样的问题？你有没有思考过其实这个问题，反映的是什么？

------------------------------------

**【我的内核到底是什么？我最核心的竞争力是什么？我的边界是什么？】**我所学所思，构建了属于我自己独一无二的壁垒，我的衣柜是属于我的，服务于我的，并不需要有多少高级的衣服，更应该舒服，舒适，一目了然。

**想不想，比想清楚，重要**

**内核和边界，也可以参考巴菲特能力圈**

"对于大多数投资者而言，重要的不是他知道什么，而是他明白自己不知道什么，只要能够尽量避免犯重大的错误，投资人**只需要做很少几件正确的事情**就足以成功了。"

有了这两个内容，我就可以给未来3年甚至5年的自己，建立一个非常完善的**个人成长知识树（衣柜变成衣帽间）**。然后根据这个专题去划分成长专题，这个时候我就可以很好的区分自己的板块了。

  -------------------------------------------------------------------------- --------------------------------------------------------------------------

我开始围绕自己想成长的领域和话题，自己去搜集内容，搭建自己的体系。

**写一个书的开篇，自己写，这个东西需要赋予你什么价值？**

**只要你开始做这件事，它一定会非常困难，但一旦你尝试去结构化自己的知识体系，做一二级的分类时，你就会发现，你开始拥有了自己的内核，而且它真的能让你非常爽！**

而当你想要去做一件事，比如说知识付费创业，你就知道你接下来需要去锻炼的能力和你的边界，这个时候在重新去围绕你想创业的内容构建知识体系，就更容易做到清晰，做到结构化了。

我开始做这个东西的时候，用了几乎快要两年的时间，时至今日我依然在**整理我的衣柜**，但是我现在的内心有一种安全感，因为我看得到我自己在**知行这个环节是合一的。**

  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  【思考】有时候我自己可能就看到一本书的书名**我觉得有意思**（或者别人送的一本书），我就放进我的知识库里面了，但是这本书我都没有读过，笔记也没有写过，**你们觉得这个东西有价值吗？**
  ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

------------

**我个人的答案是：非常有价值，**有时候把书名放在笔记里的价值，甚至比我读它还有用。

**看到了一个非常得体非常酷炫，非常贵气的领结**

为什么？**因为这个书可能来自于某个行业的大佬推荐**，或者某个合作伙伴的赠送，他告诉我如果我遇到了某个环节的问题，我直接读就可以了，那我当下虽然不需要，但是这本书现在就躺在我的知识系统里面，如果我需要的时候，我不用再去重新找，**而且它可以是我和这个大佬的沟通桥梁。**

  --------------------------------------------------------------------------------------------------------------------------------------------------
  所以，大家一定要认识到，知识系统，不是我们的任务管理器；不是你放进去了就一定得是一个笔记，是一个心得，不需要，无所谓，**你的衣柜你自己说了算。**
  --------------------------------------------------------------------------------------------------------------------------------------------------

这个知识系统，是完全服务于你自己的，是构成你对某个领域理解的重要组成，当你有任何在这个领域的时候，你就可以快速的查漏补缺。

**也附送大家一个小tips：**

现在社会，大部分人容易信息茧房的核心原因**，在于算法对知识体系的破坏。**

这个社会逐渐有一种不断膨胀的状态，大家喜欢就点赞，不喜欢就划走，当你知识体系存在漏洞的时候，这种算法会让你逐渐迷失被信息所包裹，只看得到你想看的，却并不一定是真实或者正确的信息。就好像你喜欢杀马特然后以为杀马特就是主流打扮一样。

+---------------------------------------------------------------------------+
| **所以我们要养成给文章打分的习惯，比如70-99分，低于70不用收录没有意义；** |

**第如何成为第二类收纳大师？记住三个原则：**

**全部整理、结构清晰、方便取用**

**第一个原则：全部整理**

很多人对所谓的『知识』有**很强的包袱感**，希望每一篇内容都很完整，清晰。常常因为不够完整可能就一两句话就不整理了或者堆叠在其他内容中一起。

**做过的举手1，创建超级多个学习用文件夹，然后其中一大半全部废止，没有然后**

但是这种操作其实非常浪费，要知道：**真传一句话，假传万卷书；很多你放弃的一两句话说不定就是真正有价值的知识。**

不要有负担，这个衣柜只属于你，中心思想就一个逻辑：形式和来源都不重要，图片也好句子也好哪怕一段歌词都行，只要可能对你未来有用的、你觉得错过很可惜的，就直接扔进去。

**第二个原则：结构清晰**

不要像丢垃圾一样整理你的衣服，一直堆叠一直堆叠只会混乱不堪，**最好设置分级，设置分类，打好标签。**

  ------------------------------------------------------------------------- ------------------------------------------------------------------------- -----------------------------------------------------------
        
  ------------------------------------------------------------------------- ------------------------------------------------------------------------- -----------------------------------------------------------

如果你刚开始做知识管理，**也不要一上来就创建几十个文件夹，想着全部填满，不要给自己上压力，一开始只要整理自己确定性的几篇内容就好了。**

**尽量搜集符合自己思考习惯的，跟自己有默契的内容！**

还原真实你自己衣柜的整理环节，衣柜的用法是仅服务于你的，所以当你开始去做并且养成一定使用习惯之后，自然而然会知道要怎么新增内容了。

**第三个原则：方便取用**

**就和衣服一样，你经常要用到的应该放在最顺手的位置，所以你的内容也应该是你常用的在前，不常用的在后。**
