**【工具三】：有效选择**

大部分人都有过被"割韭菜"的经历，亦或者在一些做不起来的项目上花费了大量的时间、金钱。或者学了很多自己根本不会使用的知识。脑袋一拍就决定做一个项目，结果在all in之后，却没拿到结果。

**前言：认识ROI**

很多人都听过ROI，在投资领域和营销领域中，ROI的意思是：**投入产出比（Return on Investment）**

**常见的计算方法：投入产出比=成交额/花费成本；**

比如在信息流投放的时候，很多人就会非常关注投产比，我花了多少钱，带来了多少转化率，赚了多少。

我们把这个ROI决策方法，叫做**『投资ROI』**

但是绝大多数人，哪怕到真正去打工了，除非是职业原因否则基本不会长时间接触ROI，一年到头也用不到几次。

另外一种大家可能长期接触的ROI，比如你进了一家互联网大厂、创新业务的公司，就会经常听到开会提到的『ROI』、或者投产比，这个其实不是投资ROI，而是第二种：**『选择ROI』**。

做任何一个复杂的选择之前，**考虑一下划不划算**，这个决策要素不是简单的财务指标，甚至不能量化；比如要不要做XX；要不要放弃XX；要不要买XX；我要不要做直播？我要不要做IP？等等......一系列这种复杂的选择。

所以这个就是我们今天要主讲的内容**『选择ROI』，你在本课后续内容中听到任何的ROI都是『选择ROI』**

**【收益】**

小挣青年**学习型组织**的氛围打造

小林讲课能力的不断提升和知识沉淀

**答案还不明显吗？还需要我解释吗？**

当然是做！

**所以我们应该怎么样去列举选择的宽度？**

**遇到重要的事情，只要你不是马上拍脑袋去做决定，其实都能够列出来你的收益和你的成本，大部分时间只是我们没有花时间去写下来而已，当你列下来的时候，能解决你80%以上的决策问题。**

**其次是深度：你的选择模型是否尽量具体**

深度决定了你ROI评估中能不能更准确，成功率更高的判定。

所以什么是深一点？

**在举个真实的案例：**

社群中有一位林同学，大厂出身的运营，有个G总就找到这个林同学，**计划一起合伙干一个项目**，

G总带流量，**一个月可以打一百多万的粉**，林同学负责运营和转化，合伙成立公司，**G总占70%**，**林同学占30%；**听上去挺不错的。

**我们如何用ROI的方法来进行评估？**

带流量的量（赚到的钱）：**1200万/年；**

项目**最好的情况**毛利率是30%，也就是**360万的毛利**

我们在看看按照股权分配，**最好的情况，**360万，**林同学就是分到108万。**

然后G总是只带流量，其余的**运营成本人员工资奖金等等杂费都是需要林同学这边承担的。**

**七七八八扣完大概在50-60万的毛利。**

林同学的风险是什么？如果流量带的不够，运营成本是**固定的支出，大量风险的成本**也是林同学承担；

综上所述，风险成本太大，且存在很多隐形成本，实际收益可能低于预期。

**所以林同学没有接受这次项目合作的建议。**

还有一个案例，也是餐饮行业的一个老哥，他在县城开了一家牛肉汤粉店，他和妻子争吵要不要把自己店面的门换成电动的卷帘闸门，他现在是那种手动的铁闸门。

  ------------------------------------------------------------------------------------ -------------------------------------------------------------------------------------

他们争吵不下，然后老板的妻子拿出来纸笔，说：

**来全部算成人民币，然后换成工资，你看看这次投入多少年可以赚回来。**

然后简单的一段数学题之后一拉：结论是**1年。**

当时本身这个店铺周边就在装修，打算打造一个商圈，如果说做一个门要1年时间回本，**太不划算了！**

所以这些小的案例里面体现出来的，**很多时候我们的犹豫不决，是因为分析的深度还不够。**

把深度的习惯放到一个图片内，就可以看到一个**很直接的对比：**

X型：除了少数涉及到钱的计算很严谨，大部分时间拍脑袋

Y型：除了不重要的事情拍脑袋，其他事情也可以花少量时间来评估

我们绝大多数人其实不会频繁用到投资ROI，但是同样的一个选择，**如果你能做到更深的程度，一定会比你直接拍脑袋更好**，因为你每一次的选择质量是不断在提高的，哪怕定量做的不精准，也比你直接拍脑袋来做的决定要高质量。

**和五步法假设的理论一样：拆不拆，很重要，拆的好不好，其实反而没有那么重要**

+---------------------------------------------------------------------------+
| 所以：                                                                    |
|                                                                           |
| **如果不重要的决策，拍脑袋无妨；**                                        |
|                                                                           |
| **如果要谨慎，就尽量把关键清单列出来，排一下优先级；**                    |
|                                                                           |
| **如果重要而且你犹豫不决，可以尝试把关键的项目定量，再做判断；**          |
|                                                                           |
| **如果重要且仍然犹豫不决，可以考虑做一个严格的定量公式，甚至采用投资ROI** |
+---------------------------------------------------------------------------+

这个需要长期的练习，不用一开始追求什么都往定量公式走，大部分时间，哪怕你只能做到优先级确认，也会大大提升你的选择质量。

**最后，说说高度：**

大部分时间，拉个单子就能确认了，**但是一些很复杂的选择**，往往会牵扯到四个新的角度：

**长期视角、公司视角、机会成本、时间窗口。**

**长期视角（知识管理同样适用）：**

同样的一件事，或许站在1次、1年、5年的尺度上看，ROI的评估是完全不同的。

举个大家可能都接触过的例子，**我们可能会每天花30分钟和喜欢的人（没在一起之前）聊天**，或者有时候就是闲聊散步啥也不干浪费时间，或者就是纯尬聊我们也觉得有意义，那这个意义来源是什么？

是我们考虑到当前的付出，对后续如果我们**在一起了这件事**的**收益**来看，值得！

切换成业务视角，如果你每天花30分钟的工作时间来定期做复盘和整理，这个习惯在当前你是每天付出了30分钟，但是长期收益来讲是一个培养复盘认知的习惯，是长期复利的。

**长期视角，往往会考虑的维度：成长、习惯培养、持续回报、复利等因素。**

**接下来是公司视角：**

很多时候，目前的打工人或者在职场努力的人，你看上去是你做的决策，但实际上归根结底都是公司的决策

所以经常听到很多老板在那里说你要站在老板的视角看待业务问题，这句话很多时候其实老板自己也没搞明白想要表达什么，其实背后的意义就在于，如果你站在老板的角度，你去考虑ROI的时候会很不一样，就像我们的第一个例子。

而这个视角出发考虑的ROI，也是大部分打工人在职场晋升拉开差距的关键。

**那么代入老板的角度，我们从公司视角，往往会考虑的维度：壁垒、现金流、关键风险、融资计划、品牌等等问题。**

**第三个，机会成本：**

有一句特别重要的话：**机会成本是放弃的最大代价。**

什么是机会成本？

一句话解释，你目前正有**三个白富美/高富帅追求你，你有想法在他们之中选择一个约定终身。**

当你做出选择的时候，就要放弃掉另外两个；而这个里面，可能存在有人是**渣男/渣女假扮的水货。**

**业务视角，我们采取了一个方案A而放弃B方案的时候，B方案可能取得的收益，就是A方案的机会成本。**

比如说，去不去看演唱会，**付出的实际成本可能是时间和工资**，但是如果从**机会成本**的角度，**说不定老板想给他升职就不给了。**

比如说，Luke一开始去做老带新项目的话，**付出的实际成本可能是公司资源，**但如果从**机会成本的角度来看**，可能公司更重要的一个项目他就没有精力去做了，更重要的业务就没有人管了。
