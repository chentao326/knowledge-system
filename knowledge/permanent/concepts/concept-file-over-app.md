# Definition

所有东西尽量落成普通文件（Markdown），避免锁在单一应用里。文件是跨工具、跨平台、可版本控制的最小公分母。

# Why It Matters

应用会消失、API 会变更、平台会关闭，但 Markdown 文件几乎永远可用。文件优先确保知识资产的长期可迁移性。

# Where It Appears

- Samuel 文档三大原则之一
- Karpathy 推文中"stored as markdown and images"
